# Python Examples

These examples load the local SDK from `sdk/python/src` and read `.env` automatically from the repo root.

## Install

```bash
python3 -m pip install -r examples/python/requirements.txt
```

## Required

```bash
IMA_API_KEY="sk-xxxx"
IMA_BASE_URL="https://api.imarouter.com"
```

python3 examples/python/llm_chat.py

## Images

```bash
python3 examples/python/image_banana_pro.py
python3 examples/python/image_banana_2.py
python3 examples/python/image_midjourney_niji7.py
```

`image_banana_pro.py` now runs a two-step chain:
- step 1: `gemini-3-pro-image-preview` text-to-image
- step 2: reuse the generated image URL for image-to-image

`image_banana_2.py` also runs a two-step chain:
- step 1: `gemini-3.1-flash-image-preview` text-to-image
- step 2: reuse the generated image URL for image-to-image

`image_midjourney_niji7.py` also runs a two-step chain:
- step 1: `mj_imagine` with `--niji 7`
- step 2: reuse the generated image URL for Midjourney image-to-image

## Videos

```bash
python3 examples/python/video_sd2_text2video.py
python3 examples/python/video_sd2_image2video.py
python3 examples/python/video_kling_text2video.py
python3 examples/python/video_kling_image2video.py
```

`video_sd2_text2video.py` is for Seedance 2.0 text-to-video.
It prints status and progress on every poll.

`video_sd2_image2video.py` is for Seedance 2.0 image-to-video.
It currently uses `IMA_TEST_IMAGE_HEAD_URL` by default.
Set `IMA_TEST_IMAGE_HEAD_URL` to one of your generated image URLs before running if you want to override it.
It also prints status and progress on every poll.

`video_kling_text2video.py` is for Kling text-to-video.
It now prints status and progress on every poll.

`video_kling_image2video.py` is for Kling image-to-video.
You can reuse an image generated by the earlier image examples by setting `IMA_TEST_IMAGE_HEAD_URL` to that image URL.
It also prints status and progress on every poll.
By default it uses a single image. Only when `IMA_KLING_USE_TAIL="true"` will it send `image_tail` for head-tail mode.
