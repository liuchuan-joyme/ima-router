# SDKs

ima-router currently provides local SDKs for:

| Language | SDK Guide | Notes |
| --- | --- | --- |
| Python | [python/README.md](python/README.md) | Editable install, OpenAI/Anthropic compatible, multimodal helpers |
| Go | [go/README.md](go/README.md) | Standard library only |

## Recommendation

- Keep the repository root README focused on product overview and entry links.
- Put language-specific installation, SDK usage, and API examples in each language README.
- Keep these SDK docs close to the actual SDK source so they stay easier to maintain.
